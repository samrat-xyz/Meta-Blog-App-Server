# Meta Blog App - Server

This is the **backend server** for the Meta Blog App.  
It provides RESTful APIs to handle all blog operations: create, read, update, and delete (CRUD). The server is built using **Node.js**, **Express**, and **MongoDB**.

---

## Features

-  Create a blog
-  Read all blogs or a single blog by ID
-  Update a blog by ID
-  Delete a blog by ID
-  Simple and scalable REST API

---

## Tech Stack

- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB (via Mongoose)
- **Other Packages:** dotenv, cors, nodemon, body-parser
